# Student Performance Analysis System

## Objective
This project analyzes student data to understand how study hours and attendance affect academic performance. 
It applies data preprocessing, feature engineering, and analysis techniques to extract meaningful insights.

## Dataset Description
- StudyHours: Number of hours studied per day
- Attendance: Attendance percentage
- Marks: Student marks

## Steps Performed
- Data Loading
- Data Cleaning
- Feature Engineering
- Data Analysis

## Key Insights
1. Higher study hours generally lead to better marks.
2. Attendance significantly influences academic performance.
3. Students with low attendance tend to score poorly.
4. Combined metrics like EffortScore give better understanding.
5. Removing outliers improves accuracy of analysis.

## How to Run
pip install -r requirements.txt
python main.py
